eYRC 2021-22: Soil Monitoring Bot (SM)

Task 0

Welcome to Task 0 of Soil Monitoring Bot!
The aim of this task is to get you started with installation of required software and to learn resources related to the software.

This task is divided into three parts :
  1. Software Installation
  2. Verifying Installation
  3. Learning Resources

Note :
1. It is recommended in this theme to use Windows 10 OS.
2. We will be using Verilog HDL in the entire theme.

  1. Software Installation :
    2. In this step, you will be downloading and installing Quartus Prime 19.1 along with Modelsim and Notepad++. The detailed instructions for installing Quartus Prime and Modelsim are provided in the section Introduction & Installation. For downloading softwares you can use any of the two methods mentioned in the "Introduction & Installation" section depending upon your internet connectivity.

  2. Verifying Installation:
    3. In this section you will be verifying if you have properly installed the softwares. Start with Getting started with Quartus section and then proceed with the Verifying Installation section.     5. Upload the SM#<Team-ID>_Task0.zip file on the portal. You can view all your submissions, so kindly download your submitted file from portal to verify the above steps. If anything is missing, please change it and re-upload the SM#<Team-ID>_Task0.zip file before the deadline.
